一些结论

$rank(A) = rank(A^{T}A)$

只需说明$AX = 0$ 与 $A^{T}AX =0$同解.

若$X$满足$AX = 0$,显然有$A^{T}AX = 0$成立.

反之，若$A^{T}AX = 0$,左乘$X^{T}$,有$X^{T}A^{T}AX = 0$.

即$(AX)^{T}AX = 0$,令$Y=AX$,有$Y^{T}Y = 0$,$Y = AX = 0$.
